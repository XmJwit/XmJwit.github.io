GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持Release、Archive以及单文件，右键能复制出来的链接几乎都是符合标准的。

Release、Archive使用CloudFlare加速，而单文件则会跳转至JsDelivr

注意，不支持项目文件夹
